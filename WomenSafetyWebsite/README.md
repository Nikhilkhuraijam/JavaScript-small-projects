# Women Safety in Virtual Space - Website

A professional, accessible website dedicated to educating and protecting women in digital spaces. This is a production-ready website suitable for sale and customization for various organizations and clients.

## Features

✅ **Educational Content** - Comprehensive safety tips and best practices
✅ **Emergency Resources** - Hotlines and support organization links
✅ **Report Tool** - Confidential incident reporting system
✅ **Community Forum** - Safe space for discussions and support
✅ **Safety Quiz** - Self-assessment with personalized recommendations
✅ **AI Chat Support** - Intelligent chatbot for instant help
✅ **Blog Section** - Regular safety articles and updates
✅ **Mobile Responsive** - Works perfectly on all devices
✅ **Accessible Design** - WCAG compliant for all users
✅ **Professional Branding** - Deep blue & coral color scheme

## Color Scheme

**Primary Color:** Deep Blue (#003D82) - Trust, Security
**Secondary Color:** Coral (#FF6B5B) - Action, Attention, Support

This color scheme is deliberately chosen to be:
- Professional and trustworthy
- Not associated with AI (unlike blues and cyans)
- Warm and approachable for non-tech users
- Accessible for color-blind users

## Getting Started

### For Development

1. **Open in VS Code:**
   ```bash
   code WomenSafetyWebsite/
   ```

2. **Live Server (Recommended):**
   - Install "Live Server" extension in VS Code
   - Right-click `index.html`
   - Select "Open with Live Server"

3. **Local File:**
   - Simply open `index.html` in any web browser

### For Client Deployment

1. **Rename the brand:**
   - Update "SafeSpace" to your organization name in `index.html`
   - Update logo and branding colors in CSS if needed

2. **Customize content:**
   - Replace emergency hotlines with region-specific numbers
   - Add your organization's specific resources
   - Update blog posts with relevant content

3. **Deploy to web:**
   - Host on any web server (Netlify, Vercel, AWS, etc.)
   - All files are static HTML/CSS/JS (no backend needed yet)

## Customization Guide

### Colors
Edit in `assets/css/style.css`:
```css
:root {
    --primary-color: #003D82;      /* Change here */
    --secondary-color: #FF6B5B;    /* Change here */
    /* Other colors... */
}
```

### Organization Name
Search and replace "SafeSpace" with your organization name throughout `index.html`

### Hotlines & Resources
Update the resource cards in the Resources section with actual hotlines

### Chat Responses
Edit `assets/js/script.js` - `chatResponses` object to customize AI responses

### Blog Posts
Add more blog articles in the blog section with appropriate dates

### Hotline Numbers
Search for phone numbers in `index.html` and update with actual numbers
